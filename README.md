# TaskNinja

WIP TaskNinja is a command line task list utity program, inspired by [taskwarrior](https://github.com/GothenburgBitFactory/taskwarrior)

## Install

### From Git

```bash
git clone git@github.com:luke-goddard/taskninja.git
cd taskninja
make build
make install
```

## Local Development
### Run
```bash make run ```

### Run the unit tests
```bash make tests ```

### View the database
```bash make browse ```
